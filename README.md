# Installation Guide

1. Install Node.js \
   https://nodejs.org/en/download/prebuilt-installer/current
2. Clone the repo: \
   `git clone https://github.com/Gandlfer/cavs-1.git`
3. Change to directory: \
   `cd cavs-1`
4. Install required dependencies: \
   `npm install`
6. Run the webserver: \
   `npm run start`
